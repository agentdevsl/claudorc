Workflow Designer Wireframe

  • Theme: GitHub Dark (#0d1117)
  • Layout: 3-column (templates | canvas | inspector)
  • Flow: Click "Generate with AI" → Select skill → Preview → Generate
  • Nodes: Start (green circle), End (red circle), Skill/Command/Agent (cards)
  • AI Feature: Parses skill markdown to extract workflow steps
  • Templates: From org sync (/api/templates?scope=org)
  • Inspector: Edit selected node properties
  • Fonts: Mona Sans + Fira Code (mono)
