Design Notes

Live Interactive Dashboard

Full real-time monitoring dashboard with four key areas:

  • Monitor Bar: Active sessions, token usage, git activity, API rate limits
  • Session Cards: Grouped by project, with hover controls (approve/reject/stop)
  • Live Stream: Terminal-style output for selected session with blinking cursor
  • Session Controls: Approve tools, send input, or stop sessions

Token + Cost Tracking

Real-time token monitoring per-session and aggregate:

  • Input/output/cache token breakdown with stacked bar
  • Estimated cost calculation using API pricing
  • Animated counter for total tokens across sessions
  • Rate limit usage with progress bar

Git Activity Monitor

Tracks file changes and branch activity across sessions:

  • Files changed count in monitor bar
  • Branch badges on each session card
  • Activity tab shows commits, file edits, branch operations

Animations & Interactivity

  • Cards slide in with slideInUp animation
  • Working sessions pulse with activityPulse glow
  • Live dot pulses in header and status badges
  • Token counter animates with tokenTick
  • Blinking cursor on live stream output
  • Hover reveals session control buttons (approve/stop)
  • Selected card has accent border glow
