Design Notes — Matrix Console

Matrix Rain Concept

Full-screen terminal aesthetic inspired by the Matrix. Each session occupies a vertical column showing its live output stream. The "rain" effect comes from streaming text flowing downward with fading opacity. The focused (clicked) column expands to 3x width.

  • Columns: Equal-width vertical panes, one per session
  • Focus: Click a column to expand it (flex: 3)
  • Stream: Lines anchored to bottom, new content pushes up
  • Idle: Column content fades to 40% opacity

CRT / Cyberpunk Effects

Multiple layers create the retro-terminal feel:

  • Scanlines: CSS repeating-linear-gradient on ::before
  • Vignette: Radial gradient darkening edges on ::after
  • Flicker: Subtle flicker animation on status text
  • Glow: text-shadow with green glow on key elements
  • Color: Custom --matrix-green (#00ff41) palette with dim/faint variants

HUD Overlay Panels

Floating transparent panels overlaid on the matrix viewport:

  • Sessions: Top-right, summary of active/waiting/idle
  • Tokens: Bottom-right, large counter with breakdown bar
  • Git: Bottom-left, file changes with branch breakdown
  • Rate: Top-left, API rate limit with progress bar
  • All panels use backdrop-filter: blur and staggered hudSlideIn/hudSlideUp animations

Approval UX

When a session needs approval, its column shows a semi-transparent overlay with pulsing shield icon, tool name, and approve/reject buttons in matrix-styled UI. Uses amber color for contrast against the green theme.

Typography

Uses JetBrains Mono throughout the matrix viewport for a sharper, more modern monospace feel compared to Fira Code. The standard AgentPane sidebar retains Mona Sans for contrast — the sidebar feels like a "normal" app while the main viewport is the immersive terminal.
