Design Notes

Satellite Tracker / Ground Control concept for CLI Monitor. Each CLI session is a satellite orbiting a stylized dark globe.

  • Orbital rings = projects (color-coded by orbit altitude)
  • Satellites = sessions (speed/brightness reflects status)
  • Ground stations = project repositories (antenna icons on globe)
  • Signal beams = dashed lines connecting satellite to its repo
  • Telemetry panel = per-satellite metrics (signal, altitude/tokens, velocity, fuel/rate-limit)
  • Mission timeline = launch times and mission phase transitions
  • Comms log = recent tool calls as radio transmissions
  • Working = fast orbit + bright trail; Waiting = stationary + amber beacon; Idle = slow drift + dim
