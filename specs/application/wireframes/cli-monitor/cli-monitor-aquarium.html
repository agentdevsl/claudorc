Design Notes

Aquarium / Deep Sea Concept

The main viewport is a dark deep-sea aquarium. CLI sessions are represented as bioluminescent creatures whose appearance, behavior, and luminosity encode session state at a glance.

Creature Types by Status

  • Working: Bright green glowing fish swimming rapidly with luminous trails. Size proportional to token usage.
  • Waiting: Pulsing amber jellyfish hovering in place with trailing tentacles. Glow intensifies on pulse.
  • Idle: Dim, gray-blue creatures drifting slowly near the bottom. Subdued glow indicates low activity.

Depth Layers & Bubbles

  • Y-axis = Project: Depth zones group sessions by project, with depth marker labels on the left.
  • Bubbles = Events: Tool calls and messages rise as bubbles from creatures. Bubble size indicates event importance. Labels show tool names.
  • Particles: Bioluminescent particles float in the background for ambient atmosphere.

Animations & Effects

  • Water surface wave animation with subtle caustic overlay
  • Fish swim with translateX + sine-wave vertical movement
  • Jellyfish pulse vertically with scale oscillation
  • Bioluminescent glow pulsing on all creatures
  • Sonar-ping rings animate when new sessions appear
  • Bubbles rise with horizontal drift and fade out
  • HUD panels use glass-morphism with backdrop blur

HUD Overlay Structure

  • Top bar: Session count, active count, total tokens, estimated cost, live indicator
  • Bottom bar: Selected session detail panel (click a creature to inspect)
  • Side panel: Event log as a rising bubble trail with tool names and timestamps
