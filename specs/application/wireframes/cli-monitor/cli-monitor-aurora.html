Design Notes

Concept: Aurora Borealis Command

The dashboard IS the northern lights. Aurora curtains are not decoration -- they ARE the data visualization.

Curtain height encodes token throughput. Taller curtains = more tokens consumed.
Curtain color encodes status. Green/cyan = working. Amber/pink = waiting. Gray wisps = idle.
Movement speed encodes turns per minute. Faster rippling = higher activity rate.
Constellations are completed tasks. Each star is a tool call; lines connect them into task shapes.
Shooting stars are real-time events (tool invocations, messages) streaking across the sky.
Magnetometer maps aggregate activity to geomagnetic scale: Quiet to Storm.
Observatory metrics use aurora-forecasting metaphors: Solar Wind (throughput), Kp Index (activity), Visibility (session count), Temperature (rate limit).

All HUD panels use glass-morphism (backdrop-filter: blur) over the live aurora sky. 4 sessions across 3 projects shown with realistic data.
