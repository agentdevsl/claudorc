Design Notes

Concept: Cyberpunk Neon Metropolis -- The monitoring dashboard reimagined as a bird's-eye view of a neon-lit cyberpunk city at night.

Buildings = Projects. Height encodes total token usage. Lit windows indicate active sessions. Neon trim colors map to project identity: #58a6ff agentpane, #3fb950 api-server, #a371f7 docs-site.

Drones = Sessions. Green drones fly along paths (working), amber drones hover (waiting for approval), dim drones are landed (idle). Trail effects show movement history.

HUD Panels use angular cyberpunk styling with scan lines, neon borders, and CRT effects. The bottom circuit-board trace shows real-time event flow as light pulses.

Effects: CRT scan lines, rain with neon reflections, holographic billboards, neon sign flicker, vignette darkening, data traffic light trails.
