Design Notes — Mission Control

Orbital Visualization Concept

Inspired by NASA/SpaceX mission control interfaces. Sessions are represented as orbital nodes around a central status hub. Each node's visual intensity (glow, ripple effects) communicates activity level at a glance.

  • Central hub: Aggregate stats — session count, total tokens
  • Concentric rings: Slowly rotating orbital tracks with dashed borders
  • Node glow: Working = green glow + ripple, Waiting = amber float, Idle = dim
  • Hover reveal: Detail panel slides out with goal, output, and action buttons

Grid + Scanline Atmosphere

Subtle CSS grid overlay and horizontal scanline effect create a technical monitoring aesthetic without distracting from the data. Radial gradient backgrounds add depth to the canvas.

Telemetry Strip

Bottom HUD-style telemetry bar with frosted glass effect (backdrop-filter: blur) shows key metrics. Cards animate in with staggered fadeSlideIn delays. Uses Outfit display font for large metric numbers.

Key Animations

  • coreGlow — central hub pulsing box-shadow
  • ripple — expanding rings on working nodes
  • nodeFloat — gentle vertical bobbing for waiting nodes
  • ringRotate — ultra-slow orbital ring rotation (120-240s cycles)
  • fadeSlideIn — staggered telemetry card entrance
  • dataStream — animated dash-offset on connection lines
