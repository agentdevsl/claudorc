Design Notes -- Greenhouse / Terrarium

This wireframe visualizes CLI agent sessions as growing plants inside a dark greenhouse terrarium.

  • Working sessions are tall, lush plants with animated leaf growth, green glow, and pollen/spore particles rising upward. Height encodes turn count.
  • Waiting sessions show an amber-pulsing flower bud ready to bloom (needs approval). A butterfly hovers nearby.
  • Idle sessions are wilted gray plants with mushrooms and moss at the base.
  • Root system below soil shows shared-file dependencies between sessions (dashed blue lines).
  • Weather: sun rays = token throughput, rain = events, temp gauge = API rate limit health.
  • Growth rings on trunks encode turn count like tree age rings.
  • Seed packets at bottom represent queued/upcoming tasks.
  • Fireflies drift around active plants for ambient atmosphere.
