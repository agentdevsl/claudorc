Differentiation Approaches

  • A - Glowing Dot: box-shadow: 0 0 12px with radial gradient halo creates soft glow
  • B - Left Edge: Vertical bar via ::before pseudo-element; creates visual lanes
  • C - Tinted Badge: Count badge uses color-mix() for semantic tinting; minimal dot backup
  • D - Header Gradient: Subtle 8% color mix diagonal gradient; nearly invisible but perceptible
  • E - Icon Badge: GitHub Octicons in tinted containers; semantic meaning through iconography
  • F - Bottom Border: Accent at bottom keeps header clean; grounds each column
