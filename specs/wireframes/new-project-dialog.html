Design Notes
  • Project = working context for a repo - repo selection is required
  • Project name auto-derived from repo directory name
  • Recent repos list for quick access to frequently used repositories
  • Simplified flow: select repo → add (no separate naming step)
  • Multiple users can have projects for same repo (different working contexts)
