Theme System Design Notes

  • Theme options: Light, Dark, and System (auto-detect)
  • Storage: Persist preference in localStorage with key theme
  • Implementation: Set data-theme attribute on <html> element
  • Transitions: 200ms ease-out for smooth color transitions
  • Semantic tokens: All colors use semantic names, not raw values
  • System preference: Uses prefers-color-scheme media query
  • UI controls: Segmented control (sidebar) or dropdown (settings page)
  • Keyboard: Consider ⌘+Shift+D as global toggle shortcut
