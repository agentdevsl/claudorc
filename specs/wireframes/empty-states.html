Notes

Empty State Design Guidelines

  • Hierarchy: illustration → title → action (top to bottom)
  • Messaging tone: Friendly, encouraging, never blaming the user
  • Always provide clear next action - never leave users stuck
  • Icons: 64x64px, 2px stroke, muted color
  • Secondary actions: Links or ghost buttons, never competing with primary
