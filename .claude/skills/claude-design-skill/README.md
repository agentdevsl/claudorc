# Claude Design Skill

A specialized skill for Claude Code that guides the AI toward producing professional, enterprise-grade user interfaces with intentional design direction.

## Source

Installed from: https://github.com/agentdevsl/claude-design-skill

## Purpose

This skill enables Claude to:
- Choose a design direction before writing code
- Apply consistent craft principles (4px grid, symmetrical padding, border radius consistency)
- Select appropriate design personalities based on product context
- Build interfaces with depth strategies, typography hierarchies, and meaningful color usage

## Design Personalities

The skill provides six design directions:

1. **Precision & Density** — Linear, Raycast, terminal aesthetics
2. **Warmth & Approachability** — Notion, Coda, collaborative tools
3. **Sophistication & Trust** — Stripe, Mercury, enterprise B2B
4. **Boldness & Clarity** — Vercel, minimal dashboards
5. **Utility & Function** — GitHub, developer tools
6. **Data & Analysis** — Analytics, metrics, business intelligence

## Usage

Reference the skill when building interfaces to ensure design consistency and quality.
